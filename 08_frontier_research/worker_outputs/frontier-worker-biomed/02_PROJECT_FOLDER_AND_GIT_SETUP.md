# 02 — Project Folder and Git Setup

## 2.1 Choose a fresh folder

Create a new folder so this research starts from scratch:

```text
Fresh Frontier Startup Research 2026-2030/
```

Place this package's contents into that folder.

## 2.2 Open the terminal in the folder

macOS / Linux / WSL:

```bash
cd "Fresh Frontier Startup Research 2026-2030"
```

Windows PowerShell:

```powershell
cd "Fresh Frontier Startup Research 2026-2030"
```

## 2.3 Initialize Git

```bash
git init
git add .
git commit -m "initialize fresh parallel frontier research workspace"
```

## 2.4 Confirm status

```bash
git status
```

You should see:

```text
nothing to commit, working tree clean
```

## 2.5 Do not add old research outputs

This workflow is from scratch. Do not copy old files like:

```text
scored_idea_database_repaired_2026_06_25.csv
source_evidence_ledger_old.csv
old RIDs / old top-25 memos
```

You may add your founder profile later, but the first run should be unbiased.
